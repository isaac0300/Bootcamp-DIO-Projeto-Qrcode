# 📌 Sobre o Projeto

Este projeto consiste em um gerador de QR Code para e-commerce desenvolvido com Node.js.

A aplicação foi criada com foco em escalabilidade e organização, permitindo a adição de novas funcionalidades futuramente. O sistema possibilita gerar QR Codes de forma simples e prática, podendo ser utilizado em pagamentos, links de produtos, divulgação e integração com lojas virtuais.

---

# 🚀 Tecnologias Utilizadas

- Node.js
- JavaScript
- QRCode
- Prompt-sync
- Chalk

---

# 🛠️ Conceitos Trabalhados

Durante o desenvolvimento deste projeto, foram praticados os seguintes conceitos:

- Geração de QR Code com Node.js
- Manipulação de dependências
- Estruturação de projetos por camadas
- Organização de código
- Modularização de funções

---

# 🎯 Objetivos do Projeto

Ao concluir este projeto, foi possível aprender e aplicar:

- Criação de aplicações Node.js
- Utilização de bibliotecas externas
- Desenvolvimento de aplicações escaláveis
- Separação de responsabilidades no código
- Boas práticas de organização de projetos

---

## 📚 Aprendizados

Esse projeto foi importante para reforçar conceitos fundamentais do ecossistema Node.js, principalmente relacionados à modularização, utilização de bibliotecas externas e construção de aplicações organizadas e escaláveis.

Além disso, o projeto ajudou a fortalecer a lógica de programação e a compreensão sobre arquitetura de aplicações em JavaScript.
